Настройка двухфакторной аутентификации

Для аккаунта user

Двухфакторная аутентификация обязательна для вашей учётной записи

Учётная запись помечена как EXTERNAL (внешний пользователь, контрагент или аудитор). Согласно политике ИБ, доступ к системе возможен только при включённой 2FA. Настройте её сейчас, чтобы продолжить работу. Отключить 2FA впоследствии будет нельзя.
