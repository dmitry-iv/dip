SMTP-сервер
Используется STARTTLS. Стандартный порт: 587.
Для Яндекса: id.yandex.ru/security/app-passwords, тип «Почта». Пароль сохраняется в БД зашифрованным AES-GCM.
Получатели и параметры
Большинство SMTP-серверов требуют чтобы From совпадал с логином.
Можно несколько адресов. Письма уйдут на каждый.
Когда выключено — инциденты создаются как обычно, но письма не отправляются.

Тестирование
Тесты используют сохранённые настройки. Сначала сохраните, потом нажимайте.
Подсказка по Яндекс-почте
  1. Создайте пароль приложения на id.yandex.ru/security/app-passwords, тип «Почта».
  2. В настройках почтовых программ включите «С сервера imap.yandex.ru по протоколу IMAP» и сохраните.
  3. Заполните форму выше, нажмите Сохранить, затем Проверить подключение.
  4. Если подключение OK — нажмите Отправить тестовое письмо.
