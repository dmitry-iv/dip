Подсказки
Роли
  • USER — базовый доступ (отчёт, профиль)
  • MANAGER — + журнал аудита, инциденты
  • ADMIN — полный доступ, управление пользователями
Источники
  • LOCAL — пароль хранится в системе (Argon2id)
  • EXTERNAL — для подрядчиков, обязательное 2FA, можно задать срок действия
