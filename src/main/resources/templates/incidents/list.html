Что такое инцидент?

Запись, автоматически создаваемая корреляционным движком при срабатывании одного из правил. Например: 5 неудачных попыток входа с одного IP → инцидент BRUTE_FORCE с severity 5.

Жизненный цикл

NEW → аналитик принимает в работу IN_PROGRESS → закрывается как RESOLVED или FALSE_POSITIVE.
